📅 나만의 날씨 달력 구축 가이드

기상청 데이터를 내 캘린더로 가져오는 4단계를 따라해보세요.

Step 1

저장소 복사 (Fork)

이 프로젝트의 소스 코드를 본인의 GitHub 계정으로 가져옵니다.

GitHub 저장소에서 Fork 하기
Step 2

기상청 API 키 등록

기상청 API 허브에서 발급받은 Decoding Key를 GitHub Secrets에 등록하세요.

경로: Settings > Secrets and variables > Actions > [New repository secret]
Name: KMA_API_KEY / Value: (발급받은 키 입력)
Step 3

내 지역 코드 생성

아래 버튼을 눌러 주소를 검색하면 본인 지역에 맞는 코드가 자동 생성됩니다.

# 주소를 검색하면 코드가 여기에 나타납니다.
# (기본값: 서울 중랑구)
NX, NY = 60, 127
LOCATION_NAME = "봉화산로 193"
REG_ID_TEMP = '11B10101'
REG_ID_LAND = '11B00000'

※ 복사한 코드를 update_calendar.py 파일 상단에 붙여넣으세요.

Step 4

👀 현재 달력 미리보기
